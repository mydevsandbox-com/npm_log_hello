# npm_log_hello

Simple package which outputs `hello` content to `console` and as a `return`.

```js
const { logHello } = require('@mydevsandbox-com/log-hello');
logHello();
```

```plain
> const { logHello } = require('@mydevsandbox-com/log-hello');
> logHello();
Hello from @mydevsandbox-com/log-hello
'Hello from @mydevsandbox-com/log-hello'
> 
```
